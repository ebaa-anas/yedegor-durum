# Yedegor Durum 🌯

A modern, interactive restaurant menu and ordering application built to provide a seamless user experience. 

## 🚀 Tech Stack

* **Frontend:** React, TypeScript, Vite
* **Styling:** Tailwind CSS
* **Backend / Database:** Firebase (Firestore)

## ✨ Features

* **Dynamic Menu Display:** Browse categories and menu items easily.
* **Responsive Design:** Fully optimized for both mobile and desktop screens using Tailwind CSS.
* **Real-time Data:** Fetches menu items and updates directly from Firebase.
